# 

Клонирование репозитория
git clone https://github.com/USERNAME/Advertising_platforms.git
cd Advertising_platforms


Замените USERNAME на ваш GitHub логин.

Сборка проекта
dotnet build

Запуск веб-сервиса
dotnet run

Для просмотра интерфейса добавлен Swagger, для его запуска вставьте в адресную строку браузера: http://localhost:5000/swagger/index.html
